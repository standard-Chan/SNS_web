# Firebase sns web

firebase 기반 sns web 토이 프로젝트

## 프로젝트 소개

firebase api를 이용하여 메시지, 이미지 공유하는 sns web 제작.

---------------------
## '목적 및 참고 사이트'

+ JS 기본 동기, 비동기 이해 및 활용
+ 저장 데이터 base64, blob, file 데이터 이해
